# DrAmblyo
This is an experimental VR game exploring dichoptic visual stimulation and puzzle-based engagement for amblyopia, built as a personal research and learning project
It is not a medical device and not a clinical treatment.
**Status**
Proof of concept / exploratory prototype
Actively evolving
**Ethics & intent**
This project is shared in the spirit of open research, learning, and accessibility.
It is not intended for commercial exploitation without the explicit permission of the author.
